# Instalación de librerías y su usabilidad.
## 1. Instalar las librerías necesarias
Este comando descarga e instala las "herramientas" necesarias para que Python pueda procesar tus datos:

- pandas: Es tu "Excel dentro de Python". Permite limpiar, filtrar y unir las tablas.
- numpy: Permite realizar cálculos matemáticos complejos a gran velocidad.
- scikit-learn: Es la biblioteca de Inteligencia Artificial. Aquí vive el algoritmo Random Forest que predice el riesgo.
- folium: Es el motor de mapas. Convierte tus coordenadas en el mapa interactivo HTML.
- openpyxl: Es el conector que permite a Python leer archivos modernos de Excel
```bash
pip install pandas numpy scikit-learn folium openpyxl
```
## 2. Ejecución del proyecto.
La versión que se usa en este proyecto es de python 3.13.5 para su mejor manejo para el mapeo y otras opciones.
```bash
python Process.py
```
